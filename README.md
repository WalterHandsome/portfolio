# 个人简历网站

纯 HTML/CSS/JS 实现的个人简历网站，支持深色模式、响应式布局、打印导出。

## 使用方法

1. 修改 `index.html` 中的 `resume` 对象，替换为你自己的信息
2. 推送到 GitHub 仓库
3. 开启 GitHub Pages（Settings → Pages → Deploy from branch → main）
4. 访问 `https://你的用户名.github.io/仓库名/`

## 功能特性

- 纯前端，无需构建工具
- 支持深色/浅色模式自动切换
- 响应式布局，适配手机/平板/桌面
- 支持浏览器打印导出 PDF
- 滚动淡入动画
- Tab 切换展示不同类别内容
- 技能进度条可视化
- 邮箱一键复制
- 回到顶部按钮
